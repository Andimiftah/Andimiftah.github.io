Sejarah Jong Java

Jong Java adalah organisasi pemuda yang dibentuk pada awal abad ke-20 yang bertujuan untuk meningkatkan rasa nasionalisme di kalangan pemuda Jawa. Organisasi ini memainkan peran penting dalam sejarah pergerakan nasional Indonesia.

Timeline Jong Java

1915: Pembentukan Jong Java

Jong Java dibentuk pada 7 Maret 1915 di Surakarta oleh Raden Satiman Wirjosandjojo bersama dengan tokoh-tokoh muda lainnya.

1917: Peran dalam Pergerakan Nasional

Jong Java aktif dalam memperjuangkan kemajuan pendidikan, budaya, dan politik untuk mencapai kemerdekaan Indonesia.

1928: Kongres Pemuda II

Anggota Jong Java turut serta dalam Kongres Pemuda II, yang menghasilkan Sumpah Pemuda, salah satu tonggak penting dalam sejarah perjuangan Indonesia.

1930: Perubahan Organisasi

Jong Java kemudian melebur menjadi Indonesia Muda sebagai bagian dari penyatuan gerakan pemuda dari berbagai wilayah Nusantara.
